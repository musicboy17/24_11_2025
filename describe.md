3. Программа для скачивания музыки.
   В программу загружается текстовой файл - на одной строке - одно название музыкального трека. Программа запрашивает сайты, предоставляющие возможность бесплатного скачивания музыкальных файлов в формате mp3. Если файл не находится на первом сайте - поиск на втором сайте и т.д.
   Программа запрашивает адрес для копирования музыкальных файлов. Результат - папка со скачанными файлами.

4. Идеи
    4.1. Коворкинг - CoWorking
    Идея оболочки над Windows, включающая в себя сохранение открытых программ в сеансе работы - например 1C Бухгалтерия+ MS Excel + Vlc player + MS Word + MS Edge. Сеанс сохраняет открытые окна, расположение на рабочем столе, последние сохранения файлов, если изиенения не сохранены - последнее состояние до выключения компьютера, с возможностью возвращаться к последнему - перед выключением компьютера - сотоянию. С возможностью запуска 100 сценариев, с возможностью добавления сеансов в автозапуск.

    4.2. Идея - Добавление к Коворкинг - сохранение как в видеофайле, только чтобы был не видеофайл, а запись работы приложений - с возможностью переключения между окнами - сохранение сеанса работы программ. С последующим просмотром и возможностью редактирования, записи, перезаписи - как стандартный видеофайл, переключения между приложениями.

    Английский перевод

    Ideas 4.1. CoWorking - CoWorking is the idea of a shell over Windows, which includes saving open programs in a work session - for example, 1C Accounting + MS Excel + Vlc player + MS Word + MS Edge. The session saves open windows, the location on the desktop, the last file saves, if the settings are not saved, the last state before turning off the computer, with the ability to return to the last state before turning off the computer. With the ability to run 100 scripts, with the ability to add sessions to autorun.

4.2. The idea is to add to the Coworking space - saving it as in a video file, only so that it is not a video file, but a recording of the work of applications - with the ability to switch between windows - saving the program session. With subsequent viewing and the ability to edit, record, rewrite - as a standard video file, switch between applications.
